# papers/

This directory contains preprints, working papers, and conference submissions produced from or related to the Cognitive Interview Protocol research.

## Structure

```
papers/
└── YYYY-[short-title]/
    ├── paper.pdf           ← the paper itself
    ├── paper.md            ← source in markdown (if applicable)
    └── README.md           ← abstract, venue, status, DOI (if published)
```

## Status markers

Each paper entry in its README should carry one of the following status markers:

- **Working paper** — not yet submitted for review.
- **Under review** — submitted; awaiting decision.
- **Accepted** — accepted for publication; citation details below.
- **Published** — DOI and full citation available.
