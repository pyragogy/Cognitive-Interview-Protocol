## Purpose

This example demonstrates a complete CIP-KGE pipeline run on a real node in the Pyragogy Syllabus. It shows what each artifact looks like and how they connect.

The interview transcript, evidence bundle, and Knowledge Diff in this directory are **synthetic** — constructed to demonstrate the methodology, not derived from a real interview. This is stated explicitly here and in each artifact, so that no one mistakes it for empirical evidence.

Synthetic examples are not within the scope of actual pilot sessions. They exist so that a contributor who reads the protocol can understand what the output should look like before any real session has been conducted.

## The node being modified

**Current text** (Definition section, as published at the time of this example):

> The Embodied Foundation is the developmental anchor that grounds early cognition in physical, sensory-motor inquiry rather than screen-mediated abstraction. It establishes the baseline expectation that learning requires physical manipulation and friction.

**What this example proposes:** A modification to the `definition` section and an addition to the `risk` section, based on a synthetic interview with a domain expert in early childhood development and AI-mediated learning.

## How to use this example

Read the files in order: `transcript.md` → `evidence.md` → `diff.yaml`. At each step, verify that the mapping is tight — that what the diff says the interviewee said is actually in the transcript, and that the evidence bundle correctly identifies the exchanges.

If you find places where the mapping is loose, that is a useful finding for the protocol. Open an issue.
