# Archive

This directory contains early design and design-rationale documents for the Cognitive Interview Protocol (CIP-KGE).

These files are preserved here for historical context and rationale traceability. They are **not** the operational specification of the protocol.

For the active protocol specification, refer to the consolidated document:
- [`docs/PROTOCOL.md`](../PROTOCOL.md)

## Archived Files

- **`PROBLEM.md`**: Early problem statement and design motivations.
- **`VISION.md`**: Long-term scope and direction.
- **`PRINCIPLES.md`**: Foundational design commitments (a summary table remains in the root `README.md`).
- **`PIPELINE.md`**: Original 10-stage process document (now merged into `PROTOCOL.md`).
- **`KNOWLEDGE_DIFF_SPEC.md`**: Original standalone Knowledge Diff YAML specification (now merged into `PROTOCOL.md`).
- **`INTERVIEW_GUIDE.md`**: Original standalone interview question and mapping guide (now merged into `PROTOCOL.md`).
