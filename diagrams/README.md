# diagrams/

This directory contains visual representations of the Cognitive Interview Protocol — its flow, its components, and the relationships between them.

## Naming convention

```
diagrams/
├── protocol-flow-v0.1.svg      ← the overall protocol flow
├── knowledge-diff-lifecycle.svg ← states of a knowledge diff
└── session-structure.svg       ← structure of an interview session
```

Diagrams are versioned by protocol version number. When the protocol changes substantially, a new diagram version is created rather than overwriting the existing one.

## Format

Source files (`.drawio`, `.mermaid`, `.puml`) are preferred alongside exported images, so that diagrams can be modified without proprietary tools.

## Status

No diagrams have been created yet. The first diagrams will be produced alongside the v0.1 protocol specification.
